User Study Instructions

In this user study, there are 20 pages. On each page, you will see a text prompt and results (a image and a video) generated by two AI models using that prompt. The image will contain four views (at 0, 90, 180, and 270 degrees) of the same object described by the text prompt. The video will show a rotating view (with and without color) of the object's 3D model, created from the four views. Please use both the image and the video results and answer the following questions:

  1. Which result is more 3D-Consistent? Please answer Left (L) or Right (R).
  2. Which result is better aligned with the prompt? Please answer Left (L), Right (R), or Same (S).

Definition of 3D-Consistency: the object should look the same from all four views, and the 3D model should be free of floater artifacts or stretched shapes. The colorless mesh helps you better see the shape of the 3D model. However, sometimes there could be problems with the mesh, e.g. it would generate a ground plane that doesn't exist. In these cases, please ignore the ground plane in the colorless mesh when judging 3D-Consistency.

Definition of Prompt Alignment: the object should look like the prompt.

Please record your answers using the abbreviations, e.g. "1LL, 2LL, 3LL, 4LL, 5LL, 6LL, 7LL, 8LL, 9LL, 10LL, 11LL, 12LL, 13LL, 14LL, 15LL, 16LL, 17LL, 18LL, 19LL, 20LL". Please send your response to carve3dofficial@gmail.com. Thank you for participating! Your response means a lot in our research!
